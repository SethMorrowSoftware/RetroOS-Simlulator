# PROJECT EREBUS v10.0 Walkthrough (autoexec.retro)

This is a complete progression guide for the built-in EREBUS campaign in `autoexec.retro`.

## Before you start

1. Boot the system with `autoexec.retro` active.
2. Open **Inbox** and read early messages from COMMAND/UNKNOWN.
3. Open files in `Desktop/EREBUS` as they appear.
4. Submit answers by composing an Inbox message to `command@ops.local`.
   - Subject must match the thread (`THREAD-1`, `THREAD-2`, etc.).
   - Body should contain the answer only.

---

## Enhanced presentation setup (recommended)

Before reaching THREAD-5, preload the media assets listed in `docs/required_media.md` to get the full cinematic version of the experience.
